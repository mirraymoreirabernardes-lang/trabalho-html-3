Atividade: Web Design, HTML, CSS e JavaScript

1. O que é Web Design e qual sua relação com Front-End?

Web Design é a área responsável por planejar e criar a aparência visual e a experiência de uso de um site. Ele se relaciona com o Front-End porque o Front-End transforma os layouts e ideias do design em código que funciona no navegador.

2. Três princípios de design que melhoram a experiência do usuário:

  • Consistência: elementos visuais uniformes facilitam a navegação.
  • Legibilidade: textos claros, tamanhos adequados e boa hierarquia.
  • Simplicidade: menos elementos desnecessários tornam o uso mais intuitivo.

3. Diferença entre Wireframe e Protótipo

Wireframe é um esboço simples da estrutura de um site. Protótipo é uma versão mais completa e interativa que simula o funcionamento real.

4. Três ferramentas para criar protótipos

  • Figma
  • Adobe XD
  • Sketch

5. Função da tag <header> e exemplo

A tag <header> representa o cabeçalho da página e geralmente contém o título, menus e logotipos.

<header>
    <h1>Meu Site</h1>
</header>

6. Diferença entre tag em bloco e tag inline

Tag em bloco ocupa toda a largura disponível (ex: <div>, <p>). Tag inline ocupa apenas o espaço necessário (ex: <span>, <a>).

7. Como aplicar CSS externo

<link rel="stylesheet" href="style.css">

8. Três propriedades CSS comuns

  • color: define a cor do texto.
  • background-color: define a cor de fundo.
  • margin: controla o espaçamento externo.

9. O que é DOM?

DOM é a estrutura em árvore que representa todos os elementos de uma página HTML, permitindo que o JavaScript altere conteúdo, estilos e eventos.
